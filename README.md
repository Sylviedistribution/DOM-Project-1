🛒 Shopping Cart App
📌 Description

A simple web application that simulates a shopping cart using Vanilla JavaScript.
Users can:

Add items to the cart

Remove items

Update item quantities

See the total price update in real time

⚙️ Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout

JavaScript (Vanilla) – Event handling and cart logic

🚀 Features

Add items with their unit price

Remove items with a delete button 🗑️

Increase / Decrease item quantity with + / - buttons

Real-time cart total calculation

Simple and intuitive UI
